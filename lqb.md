# 蓝桥杯全国软件和信息技术专业人才大赛（吉林大学）

蓝桥杯全国软件和信息技术专业人才大赛（以下简称：蓝桥杯大赛），由工业和信息化部人才交流中心主办、国信蓝桥教育科技股份有限公司承办。

第15届蓝桥杯大赛的赛项主要包括：
+ 软件赛（个人赛）：C/C++程序设计、Java软件开发、Python程序设计、Web应用开发、软件测试、网络安全。
+ 电子赛（个人赛）：单片机设计与开发、嵌入式设计与开发、EDA设计与开发、物联网设计与开发、5G全网规划与建设。
+ 设计赛（个人赛）：平面设计。
+ 实战赛（个人赛）：AI办公。
+ 设计赛（团队赛）：动画设计、工业产品设计、环境艺术设计、交互设计、视频设计、文创设计、其他专项。
+ 实战赛（团队赛）：人工智能。
+ 创新赛（团队赛）：数字科技。

---

## 【权威认可】

+ 竞赛奖励：2022年6月16日，吉林大学发布的《吉林大学本科学生学科竞赛体系》文件中，蓝桥杯大赛被认定为B类学科竞赛。国家级一等奖、国家级二等奖的获奖选手，将获得相应的竞赛奖励。目前奖励标准：
  - 国家级一等奖：团队赛10000元、个人赛5000元。
  - 国家级二等奖：团队赛5000元、个人赛2500元。

+ 竞赛加分：2022年8月，吉林大学计算机科学与技术学院、软件学院的《本科生推免素质类项目加分细则》文件中，蓝桥杯大赛被认定为计算机类学科竞赛。国家级一等奖、国家级二等奖、国家级三等奖的获奖选手，将获得相应的素质类项目加分。目前加分标准：
  - 国家级一等奖：0.2GPA。
  - 国家级二等奖：0.1GPA。
  - 国家级三等奖：0.05GPA。

## 【报名方式】
吉林大学的所有拟参赛学生（本科生和研究生），请加入QQ群：870408172，便于发布通知、报名组织、竞赛安排。每年10-12月组织报名，次年4-6月组织省赛和国赛，具体时间以大赛官网和QQ群通知为准。
1. 【学生认证】学生登录竞赛官方网站进行注册，由蓝桥杯大赛组委会审核学生信息（一般为几个工作日）。注：可以使用学生证、学生卡、入学通知书等有效证件进行学生认证。务必确保个人信息（学号、姓名、手机、院系、身份证号等）准确无误。院系填写学院标准全称，转专业后及时更新。
2. 【院校报名】学生登录竞赛官方网站选择科目、组别进行院校报名，由学校竞赛负责教师审核报名信息（一般为几个工作日，学校竞赛负责教师会明确通知学生报名信息通过审核或修改原因）。注：只认可院校报名（须审核报名信息），不接受个人报名。
3. 【缴纳费用】学校竞赛负责教师授权学生缴费或批量缴费。
